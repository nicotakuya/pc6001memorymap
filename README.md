# pc6001memorymap

## overview

PC-6001のメモリマップを表示するJavascriptです。

・目的のモードを瞬時に探し出せる。

・スケーリングが正確。

・ソースをちょっと直せば、表示の上下反転も可能。

## hot to run

実行例

https://nicotak.com/sonota/p6memorymap.html

Webブラウザでhtmlファイルを開くだけです。

コンボボックスから知りたいモードを選択すると、メモリマップを表示します。
