# pc6001memorymap

## overview

PC-6001のメモリマップを表示するJavascriptです。

・目的のモードをすぐ探し出せる。

・スケーリングが正確。

・ソースを直せば、表示の上下反転も可能。

## hot to run

htmlファイルを開くだけです。

コンボボックスから知りたいモードを選択すると、メモリマップを表示します。
